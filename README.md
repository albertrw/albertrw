## 🍽️ Featured Project — Digital Menu System

### 🧠 Overview
**Digital Menu** is a modern, responsive web application designed for restaurants and lodges.  
It allows guests to browse menu items, view details, and place orders digitally — enhancing user experience and service speed.

### ⚙️ Tech Stack
- **Frontend:** EJS, Tailwind CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Other:** AJAX, Dynamic Data Tables, SweetAlert, PDF Export

### 🌟 Key Features
✅ View menu items with images and details  
✅ Filter and sort menu items by name or price  
✅ Admin dashboard for adding/editing/deleting items  
✅ PDF/CSV export of menu and services  
✅ AJAX for real-time updates without page reload  
✅ Mobile-ready UI built with Tailwind CSS  

> 💡 The system was designed to support fast, secure, and real-time operations with scalability in mind — making it easy to integrate into mobile and desktop apps later.
